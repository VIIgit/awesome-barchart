Example 1: Revenue vs Orders (Different Data Sets)

Two charts showing different metrics for the same time period. Top chart shows daily revenue, bottom chart shows order count - each with its own data array.

Data: 365 days of simulated e-commerce data with different scales for each metric 
// Generate two different datasets for the same dates
const revenueData = generateRevenueData(365);
const ordersData = generateOrdersData(365);

// Create chart with per-series data
Barchart.createChart({
  container: '#multiChart1',
  // No shared data - each series has its own
  chartType: 'byDay',
  visibleWidth: 900,
  chartHeight: 180,
  barMinWidth: 6,
  scrollToEnd: true,
  timeSeries: [
    {
      data: revenueData,  // First dataset
      renderType: 'bar',
      title: 'Daily Revenue ($)',
      yAxisLabel: 'Revenue',
      barColor: '#4a90d9',
      yAxisFormat: 'K'
    },
    {
      data: ordersData,   // Second dataset
      renderType: 'bar',
      title: 'Daily Orders',
      yAxisLabel: 'Orders',
      barColor: '#27ae60'
    }
  ]
});

Example 3: Stock Analysis - Price, Volume & RSI

Financial dashboard with three different metrics: stock price (high-low), trading volume (bars), and RSI indicator (bars) - each from its own data source.

// Three different stock metrics
const priceData = generateStockPriceData(200);
const volumeData = generateVolumeData(200);
const rsiData = generateRSIData(200);

  container: '#multiChart3',

  chartHeight: 140,
  barMinWidth: 6,
  scrollToEnd: true,
  timeSeries: [
    {
      data: priceData,
      renderType: 'high-low',
      title: 'Stock Price (High-Low)',
      yAxisLabel: 'Price',
      highLowColor: '#2c3e50',
      avgMarkerColor: '#f39c12',

    },
    {
      data: volumeData,
      renderType: 'bar',
      title: 'Trading Volume',
      yAxisLabel: 'Vol',
      barColor: '#9b59b6',
      yAxisFormat: 'M'

    {
      data: rsiData,
      renderType: 'bar',
      title: 'RSI Indicator',
      yAxisLabel: 'RSI',
      barColor: '#16a085'
    }
  ]
});

Example 4: Backward Compatible - Shared + Per-Chart Data

Charts can mix shared data (from config.data) with per-chart data. The first chart uses shared data, the second provides its own.

// Shared data for multiple series
const sharedData = generateDailyData(100);
// Custom data for one specific series
const customData = generateCustomData(100);

Barchart.createChart({
  container: '#multiChart4',
  chartType: 'byDay',
  visibleWidth: 900,
  chartHeight: 150,
  barMinWidth: 10,
  timeSeries: [
    {
      data: sharedData,  // First series defines shared data
      renderType: 'bar',
      title: 'Using Shared Data',
      yAxisLabel: 'Value',
      barColor: '#4a90d9'
    },
    {
      data: customData,  // This series has its own data
      renderType: 'bar',
      title: 'Using Custom Data',
      yAxisLabel: 'Custom',
      barColor: '#e74c3c'
    }
  ]
});
