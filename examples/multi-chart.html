Example 1: Daily Values + Monthly High-Low Range

Two stacked charts sharing the same 700-day dataset. Top chart shows daily values, bottom shows monthly high-low ranges. Scroll horizontally to explore all data - notice how y-axes stay fixed!

Data: 700 consecutive days of simulated values (approximately 2 years) 
// Generate 700 days of data
const data700 = generateDailyData(700);

// Create multi-chart with shared x-axis
Barchart.createMultiChart({
  container: '#multiChart1',
  data: data700,
  chartType: 'byDay',
  visibleWidth: 900,
  chartHeight: 180,
  barMinWidth: 6,
  scrollToEnd: true,
  charts: [
    {
      renderType: 'bar',
      title: 'Daily Values',
      yAxisLabel: 'Value',
      barColor: '#4a90d9'
    },
    {
      renderType: 'high-low',
      title: 'Value Range (same data)',
      yAxisLabel: 'Range',
      highLowColor: '#e74c3c',
      avgMarkerColor: '#fff'
    }
  ]
});

Example 2: Monthly Aggregation - Three Chart Types

Same 700-day dataset aggregated by month, showing three different visualizations. Each chart has its own y-axis scale but shares the same x-axis.

Aggregation: Daily data grouped by month ‚Üí ~24 months of aggregated data 
Barchart.createMultiChart({
  container: '#multiChart2',
  data: data700,
  chartType: 'byMonth',  // Aggregate by month

Example 3: Weekly Aggregation with Scrolling

700 days aggregated by week (~100 weeks). This example shows significant horizontal scrolling with sticky y-axis behavior.

Barchart.createMultiChart({
  container: '#multiChart3',
  data: data700,
  chartType: 'byWeek',  // Aggregate by week
  visibleWidth: 900,
  chartHeight: 200,
  barMinWidth: 10,
  charts: [
    {
      renderType: 'high-low',
      title: 'Weekly Value Range',
      yAxisLabel: 'Range',
      highLowColor: '#3498db',
      avgMarkerColor: '#e74c3c',
      useThousandSeparator: true,
      yAxisStartAtZero: false
    }
  ]
});
